## 추천 사용법
- `notes/Week_{N}/` 폴더 구조를 우선 추천
- 호출 추천 포맷: `/note-taking W{N} <주제>`
  - 예시: `/note-taking W4 ingress 정리`
- 노트 작성 후 변경 저장 시 `/commit docs <요약>` 포맷 추천
- 참고 링크는 필요할 때만 웹 리서치 후 추가

### 폴더/파일명 추천
- 폴더: `Week_1`, `Week_2`, `Week_3` ...
- 파일: `YYYY-MM-DD-주제.md`
- 예시: `notes/Week_4/2026-02-22-ingress-basics.md`
